| Dato                         | Procedimento                                                                                                                                                        |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Netmask (dotted decimal)** | Prendere il formato CIDR, il numero dopo lo `/` indica il numero di 1 consecutivi della netmask (scritti dal pi√π significativo), paddare di 0 fino ad arrivare a 32 |
| **Net ID**                   | `AND` bitwise fra indirizzo IP e netmask                                                                                                                            |
| **Broadcast**                | `OR` bitwise fra indirizzo IP e `negato` della netmask                                                                                                              |
| **HostMin**                  | `NetID + 1`                                                                                                                                                         |
| **HostMax**                  | `Broadcast - 1`                                                                                                                                                     |
| **NHost**                    | $2^{32-N}-2$ con $N$ numero di 1 della netmask                                                                                                                      |
